Basic Maths properties

[a,b] - it means a and b both are inclusive (means at this time we will include value of a and b)
for example - [2,10] - {2,3,4,5,6,7,8,9,10} - the count of [2,10] is = 9 - it means a<=b
Formula - b-a+1

(a,b) - it means a and b both are exclusive (means at this time w will not include value of a and b)
for example - (2,10) - {3,4,5,6,7,8,9} - the count of (2,10) is = 7
Formula - b-a-1

s2 = 100,99,98,97,96.......1
According to the "Gause Principle" - (s1[0]+s2[0]), (s1[1]+s2[2]), (s1[3]+s2[3]) all will be return same Number
Example - (100+1=101), (99+2=101), (98+3=101), (97+4=101), (96+5=101), (95+6=101)
we can write it - 2s = 101*100 where (101-sum of each set, 100-total number of set)
2s = 101*100
s = (101*100)/2;

s1 = 1,2,3,4,5...............n
s2 = n, n-1, n-2, n-3, n-4........1
s = ((n+1)*n)/2
